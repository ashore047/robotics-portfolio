## 作品导航

### 01｜弹性折纸并联机构及制造方法

面向传统 3-RSR 并联机构摩擦损坏、抗扭能力不足，以及折纸机构依赖外置弹簧和紧固件的问题，设计了一体成型、自带弹性回复能力的 PETG/TPU 多材料柔顺并联机构。

- **核心方案：** 三条支链集成 24 个柔性铰链，以 TPU 85A 同时承担转动副运动与弹性回复功能
- **制造亮点：** 通过互锁梁界面设计和多材料 3D 打印，实现刚性支撑、柔性关节和弹性回复的单件集成
- **个人贡献：** 参与机构构型综合、双材料分区与互锁梁界面设计、切片和打印工艺调试、样机测试，并主笔技术交底书
- **成果：** 发明专利申请号 202511426156.0

[查看专利受理通知书与作品展示（PDF）](works/01-elastic-origami-parallel-mechanism.pdf)

### 02｜基于 Waterbomb 3-RSR 的多模态滚动机器人

以上述弹性折纸并联机构为核心执行体，设计并实现可在轴向压缩、可变半径转向和径向滚动三种模态间切换的滚动机器人。

- **核心创新：** 提出“形体—几何调控”思路，以柔性体连续变形在线调制滚轮姿态、接触几何与滚动动力学
- **驱动与控制：** 三路 120° 均布绳驱系统配合 OpenRB-150、舵机反馈与 IMU，实现平衡、曲率和模态管理
- **个人贡献：** 独立完成机构综合、运动学与力学建模、切片和打印工艺、控制系统搭建及多模态样机实验
- **成果：** 2025 年广东省研究生论坛“智能制造与机器人”分论坛墙报展示并获二等奖

[查看获奖学术海报（PDF）](works/02-multimodal-rolling-robot-poster.pdf)

### 03｜模块化仿人灵巧手指

在 ILDA 手指结构基础上完成复现与模块化再设计，构建具备掌指关节 2-DOF 和指间关节 1-DOF 的三自由度手指模块，实现“一根手指、多种构型”的快速重构。

- **机构设计：** 双 PSS 并联支链驱动掌指关节屈伸与摆动，交叉四连杆驱动指间关节联动弯曲
- **驱动方案：** 空心杯电机与丝杠滑台将旋转输入转化为紧凑的直线驱动
- **模块化设计：** 标准化机械/电气接口支持单指、双指、三指及不同相对位置的快速重构
- **工程验证：** 完成树脂 3D 打印骨架、金属球头拉杆、驱动与双指底座的加工装配和运动调试

[查看模块化仿人手指作品展示（PDF）](works/03-modular-humanoid-finger.pdf)
